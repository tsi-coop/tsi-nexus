How Liquid works

1

Express Intent

Type naturally or use a /command. The AI parses your input and maps it to the correct entity and action.

2

Context Surfaces

The entity's state, relationships, and full interaction history are loaded from the Sovereign Vault.

3

Govern or Analyse

Analytics actions run SQL and return an AI narrative. Mutations pass through policy guardrails before executing.
